Other Projects

In order to further develop my programming abilities in a biological context, I have completed many of the exercises on the ROSALIND bioinformatics educational platform, which provides resources and real-world exercises for developing programming skills in a biological context. Problems begin at the lowest difficulty, such as counting nucleotides on a DNA string, and are eventually built up to complex programs such as genome assembly and overlap alignment. These exercises have been pivotal to advancing my Python proficiency and introducing me to algorithm development.

Below are some highlighted scripts of mine that I selected to demonstrate my current progress. Clicking on each one will take you to the corresponding GitHub page, where you can see my code and even run the programs yourself to see how they work. If you want to see an overview of all my projects, the GitHub link can be found to the right.
